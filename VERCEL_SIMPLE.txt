📋 VARIABLES D'ENVIRONNEMENT À AJOUTER DANS VERCEL
═══════════════════════════════════════════════════════════════

Allez sur : Vercel Dashboard → Settings → Environment Variables

────────────────────────────────────────────────────────────────
1. DATABASE_URL (OBLIGATOIRE)
────────────────────────────────────────────────────────────────

Nom: DATABASE_URL
Valeur: postgresql://user:password@host:5432/database

💡 Obtenez une base PostgreSQL gratuite :
   - https://neon.tech (RECOMMANDÉ)
   - Créez un projet
   - Copiez la "Connection string"

────────────────────────────────────────────────────────────────
2. SESSION_SECRET (OBLIGATOIRE)
────────────────────────────────────────────────────────────────

Nom: SESSION_SECRET
Valeur: [Texte aléatoire de 32 caractères]

Exemple: kJ8mN2pQ4rT6vX9zA1cE3gH5iK7lM0oP

────────────────────────────────────────────────────────────────
3. NODE_ENV (OBLIGATOIRE)
────────────────────────────────────────────────────────────────

Nom: NODE_ENV
Valeur: production

────────────────────────────────────────────────────────────────
4. GOOGLE_SHEET_WEBHOOK_URL (OBLIGATOIRE)
────────────────────────────────────────────────────────────────

Nom: GOOGLE_SHEET_WEBHOOK_URL
Valeur: https://script.google.com/macros/s/AKfycbx1l-HBoUCEM9lSUJF7cdjejrXt9n_3qjo1Ox2pk1-YWL5BeUZ89rhVNp7Pcjb1NYWl/exec

────────────────────────────────────────────────────────────────
5. BLOB_READ_WRITE_TOKEN (AUTO-GÉNÉRÉ PAR VERCEL)
────────────────────────────────────────────────────────────────

⚠️ PAS BESOIN DE L'AJOUTER MANUELLEMENT !

Vercel génère automatiquement ce token quand vous utilisez Vercel Blob.
Il sera disponible automatiquement dans votre application.

🚀 COMMENT DÉPLOYER
═══════════════════════════════════════════════════════════════

OPTION 1 : Via GitHub (Recommandé)
────────────────────────────────────────────────────────────────

1. Push votre code sur GitHub :

   git add .
   git commit -m "Ready for Vercel with Blob Storage"
   git push origin main

2. Sur vercel.com :
   - New Project
   - Import votre repo GitHub
   - Ajoutez les 4 variables ci-dessus
   - Deploy !

OPTION 2 : Via Vercel CLI
────────────────────────────────────────────────────────────────

1. Installer Vercel CLI :
   
   npm install -g vercel

2. Login :
   
   vercel login

3. Déployer :
   
   vercel

4. Suivez les prompts
